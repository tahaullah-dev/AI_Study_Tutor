**AI Study Tutor** is an intelligent, web-based learning companion designed to help students study smarter. It can **summarize study materials**, **generate interactive quizzes**, and provide **quick-fire practice** challenges to boost learning efficiency and retention.

---

## ðŸŒŸ Key Features

- **AI-Powered Summaries:** Paste or upload study materials and get concise, easy-to-understand summaries.  
- **Dynamic Quiz Generation:** Generate multiple-choice, fill-in-the-blank, and true/false quizzes tailored to the material.  
- **Quick Fire Challenge:** One-minute rapid quiz sessions for focused practice.  
- **Export Options:** Save summaries and quizzes as PDFs for offline study.  
- **Customizable UI:** Theme toggle and user-friendly interface.  
- **Smart Backend:** Node.js + Express with OpenRouter AI API integration for advanced AI functionalities.

## ðŸ›  Installation

1. **Clone the repository**:  
```bash
git clone https://github.com/tahaullah-dev/AI_Study_Tutor/
````

2. **Navigate to the project folder**:

```bash
cd AI_Study_Tutor
```

3. **Install dependencies**:

```bash
npm install
```

4. **Create a `.env` file** in the root directory and add your OpenRouter API key:

```env
OPENROUTERAPIKEY=your_api_key_here
```

5. **Start the server**:

```bash
npm start
```

6. **Open your browser**:
   Visit `http://localhost:3000`
